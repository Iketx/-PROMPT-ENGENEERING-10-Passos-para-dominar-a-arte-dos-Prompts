> Aprenda prompts eficientes para tarefas de análise de dados. Domine técnicas de preparação dos dados, análise de padrões, *data visualization* e extração de *insights*.

---

## 1. Compreendendo os prompts de análise de dados.

Prompts de análise de dados precisão de uma estrutura específica para que obtenham *insghts* relevantes. A chave para isso é guiar a IA para um análise passo a passo.

> [!question] Por que uma análise de dados estruturada importa?
> * Garante qualidade aos dados;
> * Mantém o foco analítico;
> * Promove relatórios claros;
> * Facilita o processo de tomada de decisões.

## 2. Técnicas de preparação dos dados.

Quando for preparar os dados para a análise, siga o passo a passo abaixo:

PASSO 1: Avaliação inicial.
```
Revise o seguinte DATASET e:
1. Qual o tipo de dado que temos? (numérico, categórico ou séries temporais)
2. Há algum problema na qualidade dos dados?
3. Qual o tipo de preparação sera necessária para a análise?
```

PASSO 2: Construa um prompt de DATA CLEANING baseado na resposta do prompt anterior.
```
Faça uma limpeza nos dados da maneira a seguir:
1. Lide com valores ausentes.
	- Remova todos os valores faltantes ou preencha-os como nulos.
	- Explique o método escolhido.
	- Anote qualquer padrão nos dados faltantes

2. Conserte os tipos de dados:
	- Converta datas ao formato adequado
	- Garanta que numeros estão em formano numerico
	- padronize os campos de texto

3. Lidando com outiliers:
	- Identifique os valores incomuns.
	- Explique porque eles são incomuns.
	- Recomende uma maneira de lidar com esses dados.
```

PASSO 3. Crie um prompt de DATA PREPARATION. Depois da limpeza, estruture a preparação desta forma:
```
Prepare estes dados limpos por:
1. Criando novos recursos:
   - Calcule os totais mensais
   - Adicione porcentagens de crescimento
   - Gere categorias

2. Agrupando dados:
   - Por período de tempo
   - Por categoria
   - Por segmentos relevantes

3. Adicionando contexto:
   - Médias móveis
   - Benchmarks
   - Rankings
```


> [!QUESTION] Por que usar uma técnica de *data preparation*, passo a passo, é melhor? 
> * Avaliação: Previne as suposições equivocadas.
> * Limpeza. Garante uma analise fiável.
> * Preparação: Facilita a execução da análise.

## 3. Frameworks para a analise de padrões

Variados tipos de análises requerem estruturas de prompt diferentes. Abaixo, veremos como lidar com cada tipo:

**Análise estatística:**
```
Faça uma análise estatistica desse dataset:

ANÁLISE DESCRITIVA:
1. Métricas base
	- Média, mediana, moda.
	- Desbio padrão.
	- Amplitude e quartis.

2. Análise de distribuíção
	- Verifique a normalização.
	- Identifique assimetrias.
	- Anote padroẽs interessantes.

3. Detecção de outliers
	- Use a regra de 1,5 IIQ.
	- Sinalize valores pouco usuais.
	- Explique os impactos potenciais.

FORMATE OS RESULTADOS:
- Mostre os calculos
- Explique a significancia.
- Anote os problemas.
```

**Análise de tendencias:**
```
Análise, quanto tendencias, os dados, conforme os parametros abaixo:

1. Componentes em series temporais
	- Identifique sazonalidades.
	- Realce tentencias de longo prazo.
	- Anote padrões ciclicos.

2. Padrões de cresciemento
	- Calcule taxas de crescimento.
	- Compare períodos.
	- Sinalize tendencias de aceleração/desaceleração.

3. Reconhecimento de padrões.
	- Encontre padrões recorrentes.
	- Identifique anomalias.
	- Anote mudanças siginificantes.

INCLUA:
- Descrições visuais.
- Suportes numéricos.
- Explique os padrôes encontrados.
```

**Análise de cohote:**
```
ANÁLISE OS GRUPOS DE USUARIOS:
1. Pela definição de Cohote.
	- Data de entrada.
	- Primeira compra.
	- Caracteristicas do usuario.

2. Por métricas de acompanhamento.
	- Taxa de retenção.
	- Valor médio.
	- Padrões de consumo.

3. Pontos de comparação.
	- Entre "cohotes".
	- Ao longo do tempo.
	- Versus benchmarks.
```

**Análise de funil:**
```
Analise as etapas de conversão:
1. Definição de Estágio
   - Defina cada etapa
   - Defina os critérios de sucesso
   - Identifique os pontos de abandono

2. Métricas por Estágio
   - Taxa de conversão
   - Tempo em cada estágio
   - Razões de abandono

3. Foco de Otimização
   - Identificação de gargalos
   - Áreas de melhoria
   - Padrões de sucesso
```

**Análise preditiva:**
```
Analise padrões futuros:
1. Padrões Históricos
   - Tendências passadas
   - Efeitos sazonais
   - Taxas de crescimento

2. Fatores Contribuintes
   - Influenciadores-chave
   - Variáveis externas
   - Condições de mercado

3. Framework de Previsão
   - Previsões de curto prazo
   - Tendências de longo prazo
   - Níveis de confiança
```

## 4. Requisitos de visualização

Compreendendo os Elementos do Gráfico:

1. Seleção do Tipo de Gráfico POR QUE IMPORTA: Diferentes gráficos contam histórias diferentes
    
    - Gráficos de linha: Mostram tendências ao longo do tempo
        
    - Gráficos de barras: Comparar categorias
        
    - Gráficos de dispersão: Mostram relacionamentos
        
    - Gráficos de pizza: Mostram composição
        
2. Especificação do Eixo POR QUE IMPORTA: A escala adequada ajuda a entender os dados
    
    - Eixo X: Geralmente tempo ou categorias
        
    - Eixo Y: Geralmente medidas
        
    - Considere o ponto de partida (zero vs. mínimo)
        
    - Pense em quebras de escala para outliers
        
3. Escolhas de Cor e Estilo POR QUE IMPORTA: Torna a informação clara e acessível
    
    - Use cores contrastantes para comparação
        
    - Cores consistentes para itens relacionados
        
    - Considere a acessibilidade para daltônicos
        
    - Combine as diretrizes da marca, se aplicável
        
4. Elementos Necessários POR QUE IMPORTA: Ajuda os leitores a entender o contexto
    
    - Os títulos explicam o ponto principal
        
    - Os rótulos esclarecem os pontos de dados
        
    - As legendas explicam as categorias
        
    - As notas fornecem contexto
        
5. Destacando Pontos Importantes POR QUE IMPORTA: Guia a atenção do espectador
    
    - Marque mudanças significativas
        
    - Anote eventos importantes
        
    - Destaque anomalias
        
    - Mostre limites

**Solicitação vaga:**
```
Faça um gráfico dos dados de vendas.
```

**Solicitação de gráfico estruturada:**
```
Faça um gráfico dos dados de vendas.

ESPECIFICAÇÕES DO GRÁFICO:
1. Tipo de gráfico: Gráfico de linha

2. Eixo X: Linha do tempo (mensal)

3. Eixo Y: Receita em USD

4. Séries:
   - Linha do Produto A (azul)
   - Linha do Produto B (vermelho)
   - Média móvel (pontilhada)

ELEMENTOS NECESSÁRIOS:
- Posição da legenda: superior direito
- Rótulos de dados em pontos-chave
- Indicadores de linha de tendência
- Anotação de pontos de pico

DESTAQUE:
- Pontos mais altos/mais baixos
- Tendências significativas
- Padrões notáveis
```

## 5. Extração de insights

```
Extraia insights dessa análise utilizando esse framework:

1. Descobertas Chave
	- Top 3 de padrões significativos
	- Anomalias notáveis
	- Tendências críticas

2. Impactos no negócio
	- Implicações na recceita
	- Consideração de custos
	- Oportunidades de crescimentos

3. Pontos de ação
	- Ações imediatas
	- Estratégias de médio prazo
	- Recomendações de longo prazo
	
FORMATO:
Cada descoberta deve incluir:
- Comprovação por dados
- Contexto do negócio

```

## 7. Técnicas de Análise Avançada

Análise avançadas ultrapassa os padrões básicos para encontrar insights profundos. Pense que é como ser um detetive, você está a procura de pistas e conexões que não são imediatamente óbvias.

### Análise de correlação:

Essa técnica auxilia a compreender como diferentes coisas estão conectadas. Por exemplo, o clima afeta as suas vendas? Certos produtos vendem melhor juntos?
```
Aálise a relação entre as variáveis:

1. Correlações primárias
	Exemplo: Vendas vs Clima
	- Há uma relação direta?
	- Quão forte é a conexão?
	- Isso é positivo ou negativo?

2. Efeitos Secundários
	Exemplo: Clima → Fluxo de Pessoas → Vendas
	- Quais fatores conectam essas variáveis?
	- Há influências ocultas?
	- O que mais pode estar envolvido?

3. Indicadores de Causalidade
	- Quais evidências sugerem causa/efeito?
	- Quais outras explicações existem?
	- Quão certo estamos?
```

### Análise de Segmentação:

A segmentação é utilizada para agrupar informações similares afim de encontrar padrões, como ao segmentar consumidores em grupos baseados em seu comportamento de consumo. 
```
Através dos dados, seguemente:

CRITÉRIOS:
1. Segmentos primarios
	Exemplo: Grupos de consumo.
	- Alto consumo (>$1000/mes)
	- Médio consumo ($500-1000/mes)
	- Baixo consumo (<$500/mes)

2. Sub-seguementos
	Para cada grupo, análise:
	- Frequencia de compras
	- Produtos preferidos
	- Reatividade as promoções

FORMATO DE RESPOSTA:
- Perfil detalhado de cada grupo
- Mensure e quantifique os seguementos
- Oportunidades de crescimento
```


### Análise da Cesta de Mercado:

Compreenda quais itens são adquiridos em conjunto.
```Análise os padrões de compra:
1. Combinações de itens
	- Pares frequentes
	- Agrupamentos comuns
	- Combinações não usuais

2. Regras de associação
	- Métricas de suporte
	- Níveis de confiança
	- Cálculos de elevação

3. Aplicações ao negócio
	- Posicionamento de produto
	- Sugestões de combos
	- Planejamento de promoções
```


#### Detecção de Anomalias:

Encontre padrões estranhos e *outliers*:
```
Análise desvios:
1. Definção de padrões:
	- Comportamento esperado
	- Intervalos esperado
	- Sazonalidade

2. Análise de desvios:
	- Mudanças significativas
	- Combinações atipicas
	- Padrão temporal


3. Avaliação de impactos
	- Relevancia para o negócio
	- Análise da causa raíz
	- Precauções
```


> [!question] Por que a técnicas de análise avançadas importam?
> * Encontra padrões ocultos;
> * Revela insights mais profundos;
> * Sugere novas oportunidades;
> * Prediz tendencias futuras.

## 8. Armadilhas comuns:

1. ** Problemas de clareza **
	-  Métricas vagas
	-  Agrupamentos mal definidos
	-  Cronogramas ambíguos
2. ** Problemas de estrutura **
	-  Tipos de análises misturadas
	-  Prioridades mal definidas
	-  Formatos inconsistentes 
3. ** Lacunas contextuais ** 
	-  Contexto ausente
	-  Objetivos incertos
	-  Escopo limitado

## 9. Guias de Implementação

1. ** Comece com metas claras **
	-  Estipule objetivos
	-  Defina métricas
	-  Estabeleça contexto
2. ** Estruture sua análise **
	-  Use frameworks
	-  Siga padrões
	-  Mantenha a consistência
3. ** Valide Resultados **
	-  Confira os cálculos
	-  Verifique padrões
	-  Confirme as conclusões
